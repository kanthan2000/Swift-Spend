# Expense Tracker

A modern, beautifully designed React Native application for tracking personal income, expenses, and managing financial goals.

## Features

- **Dashboard**: Get a quick overview of your total balance, recent transactions, and goal progress.
- **Dynamic Reports**: Visualize your spending habits with donut charts and monthly comparison metrics.
- **Transaction Management**: Add income or expenses quickly, assign them to categories, and track them across multiple bank accounts.
- **Multi-Account Support**: Manage multiple accounts (e.g., checking, savings, credit cards) and view balances individually or combined.
- **Custom Categories**: Create your own customized categories with custom icons and colors for precise tracking.
- **Interactive Modals**: Filter your income and expense details effortlessly with dynamic category filter pills.

## Getting Started

### Prerequisites
- Node.js
- React Native CLI
- Android Studio / Xcode (for emulation and building)

### Installation

1. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

2. Run the application:

   **For Android:**
   ```bash
   npx react-native run-android
   ```

   **For iOS:**
   ```bash
   npx pod-install
   npx react-native run-ios
   ```

## Tech Stack
- **Framework**: React Native
- **Language**: TypeScript
- **State Management**: Zustand
- **Storage/Database**: SQLite (react-native-sqlite-storage)
- **Icons**: react-native-vector-icons
- **Charts**: react-native-svg
